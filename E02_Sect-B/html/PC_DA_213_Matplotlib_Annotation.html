정렬 제어¶

수직 정렬 옵션
• center : 텍스트 상자의 가운데와 관련 있다.
• top : 텍스트 상자의 위쪽과 관련 있다.
• bottom : 텍스트 상자의 아래쪽과 관련 있다.
• baseline : 텍스트의 기준선과 관련 있다.

수평 정렬 옵션
• center : 텍스트 상자의 가운데와 관련 있다.
• left : 텍스트 상자의 왼쪽과 관련 있다.
• right : 텍스트 상자의 오른쪽과 관련 있다.

경계 상자 제어¶

bbox 파라미터
• facecolor : 상자에 사용하는 컬러
• edgecolor : 상자 모양의 에지에 사용하는 컬러
• alpha : 상자와 배경을 혼합하기 위해 투명도를 설정할 때 사용
• boxstyle : round와 square 중 하나로 상자 스타일을 설정
• pad : boxstyle을 square로 설정할 때, 텍스트와 상자 옆 사이를 채울 용량을 정의

In [5]:
# pyplot.text() 함수는 딕셔너리를 입력으로 취하는 bbox 파라미터를 지원한다.
import numpy as np
import matplotlib.pyplot as plt

X = np.linspace(-4, 4, 1024)
Y = .25 * (X + 4.) * (X + 1.) * (X - 2.)

box = {
    'facecolor' : '.75',
    'edgecolor' : 'k',
    'boxstyle'  : 'round'
}

plt.text(-0.5, -0.20, 'Backmard minimum', bbox=box)

plt.plot(X, Y, c='k')
plt.show()

# 그림에 모양을 추가하는 방법
import matplotlib.patches as patches
import matplotlib.pyplot as plt

# 원
shape = patches.Circle((0, 0), radius=1., color='.75')
plt.gca().add_patch(shape)

# 직사각형
shape = patches.Rectangle((2.5, -.5), 2., 1., color='.75')
plt.gca().add_patch(shape)

# 타원
shape = patches.Ellipse((0, -2.), 2., 1., angle=45., color='.75')
plt.gca().add_patch(shape)

# 장식상자
shape = patches.FancyBboxPatch((2.5, -2.5), 2., 1., boxstyle='sawtooth', color='.75')
plt.gca().add_patch(shape)

# 모두 표시
plt.grid()
plt.axis('scaled')
plt.show()

# 표준 맷플롯립 눈금 API를 사용해 막대 차트의 눈금을 설정하는 작업
# ticker.Locatoer : 눈금의 위치를 생성
# ticker.Formatter : 눈금용 레이블을 생성
import numpy as np
import matplotlib.ticker as ticker
import matplotlib.pyplot as plt

name_list  = ('Omar', 'Serguey', 'Max', 'Zhou', 'Abidin')
value_list = np.random.randint(0, 99, size=len(name_list))
pos_list   = np.arange(len(name_list))

ax = plt.axes()
ax.xaxis.set_major_locator(ticker.FixedLocator((pos_list)))
ax.xaxis.set_major_formatter(ticker.FixedFormatter((name_list)))

plt.bar(pos_list, value_list, color='.75', align='center')
plt.show()

# 눈금 API의 핵심이 레이블에 관련한 축약어
import numpy as np
import matplotlib.ticker as ticker
import matplotlib.pyplot as plt

def make_label(value, pos):
    return '%0.1f%%' % (100. * value)
    
ax = plt.axes()
ax.xaxis.set_major_formatter(ticker.FuncFormatter(make_label))

X = np.linspace(0, 1, 256)
                             
plt.plot(X, np.exp(-10*X), c='k')
plt.plot(X, np.exp(-5*X), c='k', ls='--')
plt.show()

# 눈금 API의 핵심이 레이블에 관련한 축약어
import numpy as np
import datetime
import matplotlib.ticker as ticker
import matplotlib.pyplot as plt

start_date = datetime.datetime(2016, 1, 1)

def make_label(value, pos):
    time = start_date + datetime.timedelta(days = 365 * value)
    return time.strftime('%b %y')
    
ax = plt.axes()
ax.xaxis.set_major_formatter(ticker.FuncFormatter(make_label))

X = np.linspace(0, 1, 256)
                             
plt.plot(X, np.exp(-10*X), c='k')
plt.plot(X, np.exp(-5*X), c='k', ls='--')

labels = ax.get_xticklabels()
plt.setp(labels, rotation=30.)
plt.show()
